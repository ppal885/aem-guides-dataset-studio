## Baseline UAC Focus
- Use evidenced baseline mechanics: versioned topics/maps/assets, docstate, metadata snapshot, restore/update, export, and output generation from selected baseline.
- Watch for stale metadata, latest-content leakage, or versioned asset mismatch only when evidence names the affected entity/output.
- Validate export behavior only when Jira mentions baseline export, packaged output, or downstream artifact state.
- Ask which baseline version, docstate, asset version, and metadata snapshot are authoritative when missing.
