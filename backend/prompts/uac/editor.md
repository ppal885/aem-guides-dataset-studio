## Editor UAC Focus
- Use evidenced Web Editor flows: context menu, toolbar action, selection persistence, paste image, undo/redo, save/reload, warning display, preview, and synchronization.
- Include read-only content behavior only when Jira mentions locked/reused/read-only content, permissions, or non-editable fragments.
- For paste image, validate only the evidenced upload/rendition/reference/output behavior; do not add generic media checks.
- Tie every UI scenario to a concrete action, entity, role, output, or similar Jira key.
- Ask which authoring workflow, role, browser/editor mode, and expected persistence behavior are in scope when missing.
