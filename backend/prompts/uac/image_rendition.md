## Image Rendition UAC Focus
- Use evidenced rendition mechanics: dc:format, MIME type, source extension, vector files, .ai/.eps/.svg handling, thumbnail/rendition generation, cache refresh, cropping, and output parity.
- Validate rendition generation only for the format, path, output, or asset entity named in current or similar Jira evidence.
- Treat missing/stale rendition, wrong dimensions, image scaling, or vector conversion as separate risks only when evidence names them.
- Ask for required rendition format, MIME type, dimensions, or conversion expectation when Jira omits it.
- Avoid broad visual QA unless a concrete image/rendition signal exists.
